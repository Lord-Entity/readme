### Technical Profile
Computer scientist by trade, problem-solver by nature. I spend my time turning complex logic into digital experiences that people actually enjoy, not just tolerate. I build for impact, code with purpose, and always keep the end-user in the loop. I prioritize architectural integrity and performance, ensuring that every line of code serves a specific functional purpose. From low-level systems programming to fluid, motion-augmented interfaces, my focus remains on building for impact and long-term scalability.

* Currently pursuing a Bachelor's degree at ITU.
* Specialized in reusable library development and full-stack SaaS architecture.
* Committed to bridging the gap between technical complexity and user-centric design.
